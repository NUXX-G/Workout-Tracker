# Training Tracker

Proyecto multi-plataforma para registrar entrenamientos, crear rutinas y analizar el rendimiento. Incluye **backend en Java (Spring Boot)**, **frontend web (HTML, CSS, JavaScript)** y futura **app Android**.

## Tecnologías utilizadas

- **Backend:** Java con Spring Boot, API REST
- **Frontend web:** HTML, CSS, JavaScript
- **App Android:** Java/Kotlin (pendiente de desarrollo)
- **Base de datos:** MySQL o PostgreSQL
- **Extras:** scripts SQL para crear la base de datos y datos de ejemplo

## Funcionalidades principales

- Gestión de usuarios y autenticación
- Gestión de ejercicios y rutinas
- Cronómetro entre series
- Exportación de rutinas a PDF
- API REST consumible por web y Android
